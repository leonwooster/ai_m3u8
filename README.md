# M3U8 Video Downloader

A Windows Forms application that allows you to download videos from websites that use m3u8 playlists.

## Features

- Enter a URL containing m3u8 video content
- Download videos from the URL
- Select where to save the downloaded files
- Simple and intuitive user interface

## Requirements

- .NET Framework 4.7.2 or higher
- Windows operating system

## Usage

1. Enter the URL of the webpage containing the video
2. Click "Analyze" to detect available m3u8 streams
3. Select a stream quality if multiple options are available
4. Choose a download location
5. Click "Download" to save the video to your computer
